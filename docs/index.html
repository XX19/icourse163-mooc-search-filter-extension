🎓 中国大学MOOC课程筛选器

快速筛选中国大学MOOC平台上的已结束课程，提高学习效率

✨ 主要功能

❌ 状态识别

识别已结束但无法查看的课程，帮助整理课程列表

🎯 互斥逻辑

智能按钮互斥，避免筛选冲突，操作更流畅

🚀 使用方法

  1. 访问中国大学MOOC网站课程列表页
  2. 搜索课程关键词
  3. 点击"已结束"筛选所有已结束课程
  4. 点击"已结束可查看"筛选可学习的课程
  5. 点击"结束无法查看"查看已关闭的课程
  6. 点击右侧❌图标取消筛选

💡 适用场景

  • 寻找仍可学习的已结束课程
  • 整理和管理课程列表
  • 提高课程浏览效率
  • 课程状态快速识别
