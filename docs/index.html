🎓 中国大学MOOC课程筛选器

快速筛选中国大学MOOC平台上的已结束课程，提高学习效率

✨ 主要功能

🔍 智能筛选

一键筛选已结束且可查看内容的课程，快速找到仍可学习的资源

❌ 状态识别

识别已结束但无法查看的课程，帮助整理课程列表

🎯 互斥逻辑

智能按钮互斥，避免筛选冲突，操作更流畅

📱 响应式设计

适配各种屏幕尺寸，按钮位置智能调整

🚀 使用方法

  1. 访问中国大学MOOC网站课程列表页
  2. 点击"已结束可查看"筛选可学习的课程
  3. 点击"结束无法查看"查看已关闭的课程
  4. 点击右侧❌图标取消筛选

💡 适用场景

  • 寻找仍可学习的已结束课程
  • 整理和管理课程列表
  • 提高课程浏览效率
  • 课程状态快速识别
